**HeyCoder** 是面向开发者的技术内容项目，聚焦开源项目与技术相关话题。正文在微信公众号 **HeyCoder** 更新；本仓库同步保存相同内容的 Markdown，便于全文检索、引用链接与通过 Issue / PR 参与协作。

---

## 简介

- **公众号**：日常阅读与推送，请使用微信搜一搜查找 **HeyCoder**。
- **本仓库**：与公众号对应的文稿与项目推荐见 [`content/`](content/) 目录。
